# @thepeaklab/eslint-config

## Installation

### Install package via npm or yarn

#### npm

    npm install @thepeaklab/eslint-config

#### yarn

    yarn add @thepeaklab/eslint-config

### Add package in your eslint config

    {
      "extends": "@thepeaklab/eslint-config"
    }
